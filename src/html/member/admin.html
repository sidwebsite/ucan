    112學年度_UCAN學校總管理者資料確認

    Dear學校總管理者：
    每學年將定期進行各校UCAN總管理者資料確認作業，為確保學校UCAN總管理者聯絡資料正確，以利總管理者可以成功接收到UCAN辦公室不定期所發出的最新消息，並能順利使用UCAN後台功能。勞請師長協助檢視總管理者資料是否正確。

    現任學校總管理者資料

    *請務必再次確認信箱地址是否填寫正確！

    *格式：0X-XXXXXX#XXXX

    *請確實填寫總管理者中文姓名，勿填校名！

    *學校總管理者是否有異動？

    提醒您

    1. 本次資料確認後，貴校總管理者資料將不開放修改，包含前後台帳號基本資料，若需再次調整請洽UCAN計畫辦公室。
    2. 學校總管理者的姓名，請務必填寫實際執行UCAN業務之人員(窗口)的中文姓名。
    3. 總管理者帳號需妥善保管，僅供申請者本人使用，不得借予他人使用，若帳號被非本人使用，致使個資外洩情事，申請者需自負相關損失及法律責任。
    4. 標示「 * 」為必填欄位，請確實填答。
